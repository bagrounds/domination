# Explicit game save and load controls

**Source:** README TODO list

**Status:** Open

## Description

Add explicit save/load controls for game state. Currently, game state persists automatically in local storage, but there is no way for users to manually save, name, or load specific game states.

## Potential Features

- Save game with a custom name
- Load from a list of saved games
- Export/import game state as a file
- Share saved game states with other players
